# Find product answers, release updates, and API guidance faster

D2L helps institutions and organizations create learning experiences that are accessible, personalized, and measurable. Brightspace is the core learning platform, and the broader D2L product surface extends that experience with AI assistance, interactive content creation, learning insights, outcomes tracking, integrations, and e-commerce.

## Ask AI or search the docs

If a visitor already knows the task, the Assistant or search should be the fastest path into product help, release guidance, or developer documentation.

A strong documentation hub should make it easy to move between product guidance, release communications, and developer resources without forcing visitors to guess which surface owns the answer.
