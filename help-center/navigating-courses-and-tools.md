# Navigating courses and tools

The Brightspace student documentation covers more than account access. Once learners are inside the platform, the next challenge is usually orientation: finding the right course, understanding the homepage, and knowing which tool handles which job.

## Start with the course shell

Most learners begin from one of these entry points:

- the organization homepage after sign-in
- a course list or pinned courses view
- a notification or direct link into a course

From there, learners typically need to do three things quickly:

1. Find the current course.
2. Identify where announcements, content, and due dates live.
3. Understand which tool to use for submissions, discussions, or grades.

## Good support patterns

- Route "Where do I find my course?" questions to course discovery and homepage guidance.
- Route "Where is the assignment/content/discussion?" questions to the relevant tool, not to a generic FAQ.
- Keep the top learner journeys clickable so support content behaves like a help center, not like a long article dump.
