# Subscribing to release notes

Brightspace release notes are published on a monthly cadence. The simplest subscription model is to follow the central *Release Notes Updates* discussion so each new monthly drop triggers the notification style you prefer.

## Subscribe once

{% stepper %}
{% step %}
### Sign in to Brightspace Community

Open the public community and sign in with the account you use for Community participation.
{% endstep %}

{% step %}
### Open the Release Notes Updates discussion

Navigate to the recurring discussion thread where D2L posts new monthly release-note updates.
{% endstep %}

{% step %}
### Bookmark the discussion

Use the ribbon or bookmark action on the discussion so it becomes one of your tracked posts.
{% endstep %}
{% endstepper %}

## Choose how you want to be notified

1. Open your profile menu in Brightspace Community.
2. Go to *Account & Privacy Settings*.
3. Open *Notification Preferences*.
4. Under post notifications, enable popup alerts, email alerts, or both for new comments on bookmarked posts.

{% hint style="info" %}
The discussion itself is closed to community comments, which is useful here because it reduces reply noise and keeps the thread focused on update notifications.
{% endhint %}
